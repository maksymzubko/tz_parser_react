## Requirements
``
Node - v18.13.0
``
``
NPM - v9.2.0
``

## Installation

- Install node 18 (skip if already)
- create .env file in project folder and paste your variables (template below)

## .env template
```
VITE_API_TZ=YOUR_API_LINK
```

## Running the app
```bash
# install deps
$ npm install
```
```bash
# run app
$ npm run dev
```
